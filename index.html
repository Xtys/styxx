INSTALLATION GUIDE

(After Kez patch, the method to install mods)

Step 1: Set language option in game 'settings' to kr/ ru/ cn etc (not En)

Step 2: Download mods of preference and complie using VPK Creator

Step 3: Drop complied pak file(s) into 'dota_language' folder. eg 'dota 2 beta/game/dota_koreana' directory.

Step 4: Start Dota
